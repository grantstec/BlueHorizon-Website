## Reuse For Other Designs

To add another live CAD view on any page, copy this block and replace the URL.

```html
<iframe
	src="https://cad.onshape.com/documents/YOUR_DOCUMENT_ID"
	style="width: 100%; height: 720px; border: 1px solid #d8dee9; border-radius: 8px;"
	loading="lazy"
	allowfullscreen
></iframe>
```

## Access/Permissions Note

The document must be shared with your team or made public in Onshape, or users will see a permissions/login page.

If you still see "cad.onshape.com refused to connect", that is Onshape frame protection on that document/view. In that case, use the direct Onshape link above, or publish/share a version that allows embedding.
